All of the images MyBB uses are stored in the 'images' directory in the root MyBB folder. Here you will be able to find out what each image represents, and where they are shown.

buddies.gif
Shown next to "Select from Buddies" on the compose PM page.
buddy_away.gif
Image shown to signify a user who has their account set to "Away"; used on the Thread Display and in the Buddy/Ignore List in the User CP.
buddy_delete.gif
Button allowing you to remove a member from your buddy or ignore list. Used on the Buddy/Ignore list.
buddy_offline.gif
Image shown to signify a user's status as offline; used on the Thread Display and in the Buddy/Ignore List in the User CP.
buddy_online.gif
Image shown to signify a user's status as online; used on the Thread Display and in the Buddy/Ignore List in the User CP.
collapse.gif, collapse_collapsed.gif
Used for the collapse/expand links for forum categories on the Index Page
default_avatar.gif
The avatar shown in the buddy popup if a user does not have an avatar.
dismiss_notice.gif
Button allowing you to dismiss the new private message noticed. Used on every page, when a new message is in the user's inbox (and they have alerts enabled).
dot_folder.gif
Represents a thread in which the user has posted
dot_hotfolder.gif
Represents a popular thread in which the user has posted
dot_hotlockfolder.gif
Represents a closed popular thread in which the user has posted
dot_lockfolder.gif
Represents a closed thread in which the user has posted
dot_newfolder.gif
Represents a thread with new posts in which the user has posted
dot_newhotfolder.gif
Represents a popular thread with new posts in which the user has posted
dot_newhotlockfolder.gif
Represents a closed popular thread with new posts in which the user has posted
dot_newlockfolder.gif
Represents a closed thread with new posts in which the user has posted
error.gif
Used in the inline error messages used throughout the board.
female.gif
Not used; signifies a female user.
folder.gif
Represents a normal thread (not popular, no new posts, not closed)
fw_pm.gif
Represents a private message which the user has forwarded to another user
hotfolder.gif
Represents a popular thread
hotlockfolder.gif
Represents a closed popular thread
invalid.gif
Used when a field does not validate correctly on the registration page.
jump.gif
Used beside each thread with new posts on the Forum Display page to jump to the newest post in the thread.
lockfolder.gif
Represents a closed thread
logo.gif
The MyBB logo, shown by default in the header
male.gif
Not used; signifies a male user.
minioff.gif
The miniature icon shown in front of each subforum to represent no new posts
miniofflock.gif
The miniature icon shown in front of each subforum to represent a locked forum
minion.gif
The miniature icon shown in front of each subforum to represent new posts
movefolder.gif
Represents a moved thread.
nav_bit.gif
The icon used to indicate the active navigation element in the Admin CP.
new_pm.gif
Represents a new private message
newfolder.gif
Represents a thread with new posts
newhotfolder.gif
Represents a popular thread with new posts
newhotlockfolder.gif
Represents a closed popular thread with new posts
newlockfolder.gif
Represents a closed thread with new posts
off.gif
Used on the Index Page and Forum Display to indicate a forum with no new posts
offlock.gif
Used on the Index Page and Forum Display to indicate a forum closed to postings
old_pm.gif
Represents an old private message
on.gif
Used on the Index Page and Forum Display to indicate a forum with new posts
paperclip.gif
Used on the Forum Display to indicate a thread with one or more attachments
pixel.gif
Used as a spacer
pollbar.gif
Used to make poll results graphs (middle part)
pollbar-e.gif
Used to make poll results graphs (end part)
pollbar-s.gif
Used to make poll results graphs (start part)
printable.gif
Used on the Portal and Thread Display to link to the printable version of the thread
re_pm.gif
Represents a private message to which the user has replied.
send.gif
Used on the portal to link to the form to send the thread to a friend
spacer.gif
Used as a spacer
spinner.gif, spinner_big.gif
Used when waiting for something to load via AJAX or other.
star.gif
These are the default stars shown in the postbit under the username. They represent the amount of posts the user has posted or the user's role on the board
star_rating.gif
Used for thread ratings on the Forum Display and Thread Display.
subscribe
Used on the Thread Display to subscribe to the thread.
thead_bg.gif
The background for the 'thead' class
unsubscribe.gif
Used on the Thread Display to unsubscribe to the thread (if enabled in Admin CP)
valid.gif
Used when a field validates correctly on the registration page.
closed.gif
Shown instead of newreply.gif if the thread is closed.
Used on the Thread Display page
newreply.gif
Shown as a link for users to create a new reply to the thread
Used on the Thread Display page
newthread.gif
Shown as a link for users to create a new thread in the forum
Used on the Thread Display page, and Forum Display page
pm_delete.gif
Shown as a link to delete the PM
Used in Private Messages
pm_forward.gif
Shown as a link to forward the PM
Used in Private Messages
pm_reply.gif
Shown as a link to reply to the PM
Used in Private Messages
postbit_delete.gif
Shown as a link to delete the post
Used on the Postbit
postbit_edit.gif
Shown as a link to edit the post
Used on the Postbit
postbit_email.gif
Shown as a link to email the user
Used on the Postbit and Forum Team
postbit_find.gif
Shown as a link to search for all messages posted by this user
Used on the Postbit
postbit_multiquote.gif
Shown as a link to enable Multiquote on this post
Used on the Postbit
postbit_multiquote_on.gif
Shown as a link to disable Multiquote on this post
Used on the Postbit
postbit_pm.gif
Shown as a link to send the author a private message
Used on the Postbit, Member List, Forum Team
postbit_profile.gif
Shown as a link to view the profile of a user
postbit_quote.gif
Shown as a link to quote the post in a new reply
Used in the Postbit
postbit_replyall.gif
Shown as a link to reply to all recipients and the sender in a PM
Used in the Private Messages
postbit_report.gif
Shown as a link to report the post to moderators
Used in the Postbit
postbit_spam.gif
Shown as a link to mark the post as spam (when the Akismet plugin is enabled)
Used in the Postbit
postbit_warn.gif
Shown as a link to warn the author of the post
Used in the Postbit
postbit_www.gif
Shown as a link to visit the author's website
Used in the Postbit
rateuser.gif
Shown as a link to rate the user
Used on the reputation summary
