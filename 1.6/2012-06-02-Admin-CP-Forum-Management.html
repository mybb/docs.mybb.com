Forum & Posts: Forum Management

The Forum Management section of the Admin CP allows you to manage all of the forums on your board. It allows you to control all permissions and settings for all forums.

From the home listing page, you see a 3-tiered listing of your forums: the primary categories, the forums within those categories, and the subforums within this forums. To see anything deeper, you can click on the name of a forum, which will show all child forums within that forum. From the listing page, you can manage the display order of all of the categories, forums, and first-level sub forums.

The View Forum page provides access to all permissions and settings for that particular forum, and shows all child forums/sub forums in that forum. From here, you have access to edit the forum settings, add a child forum, or copy the forum settings and/or permissions to another, or new, forum.

You can also manage the sub forums, permissions, and moderators from this page through the javascript "sub-tabs," which change content without reloading the page. On the view forum page, these are the "Sub Forums," "Permissions," and "Moderators" tabs.

The base view forum page is the sub forums page.

Forum & Posts: View Forum's Subforums

Sub forums shows a view similar to that of the home listing page, showing the forums, their sub forums, and then sub forums within those. You can also manage the display orders for these forums, and access the same options as those on the home listing page.

In this tab, you see a quick permissions table, with options to edit the full permissions for that user group. This allows you to edit permissions quickly, or go in-depth and manage all of the permissions for each user group, all from the same location.

Quick permissions allow you to quickly edit the basic forum permissions for all user groups. You can set quick permissions when adding a forum or edit them when editing a forum's settings, in addition to managing them from the View Forum page.

Can View 
Can the group see the forum? If this is disabled, all other permissions do not need to be set.
Can Post Threads 
Can the group post new threads in the forum?
Can Post Replies 
Can the group post replies in existing threads in the forum?
Can Post Polls 
Can the group post polls in the forum?
Can Upload Attachments 
Should the group be allowed to upload file attachments in this forum?
All 
This will select all permissions, setting all to "Yes"

You can also "Use Group Default," which will use the standard permissions from the user group's settings.

Forum & Posts: Full Permissions

On the Permissions view from View Forum, you will also see a Controls column, allowing you to set the full permissions (Set Permissions) or edit existing full permissions (Edit Permissions). This will take you a page that offers all permissions for each forum, not just the most commonly used permissions.

Use Permissions 
Selecting "Use user group permissions or inherit from parent forums" will use any inherited permissions, be it from the parent forum, or user group. Selecting "Use custom permissions (below)" will allow you to set the permissions for this group for this forum only.

[edit] Viewing

Can view forum 
Should users of this group be able to view this forum?
Can view threads within forum 
Should users of this group be able to view the threads within this forum? If set to No, they will be able to see the thread listing, but will not be able to view the thread itself.
Can download attachments 
Should users of this group be able to download attachments posted in this forum?

[edit] Posting/Rating

Can post threads 
Should users of this group be able to post threads in this forum?
Can post replies 
Should users of this group be able to post replies in this forum?
Can post attachments 
Should users of this group be able to upload attachments in this forum?
Can rate threads 
Should users of this group be able to rate threads in this forum?

[edit] Editing

Can edit own posts 
Should users of this group be able to edit their own posts in this forum?
Can delete own posts 
Should users of this group be able to delete their own posts in this forum?
Can delete own threads 
Should users of this group be able to delete their own threads in this forum?
can edit own attachments 
Should users of this group be able to remove their attachments after posting them in this forum?

[edit] Polls

Can post polls 
Should users of this group be able to post polls in this forum?
Can vote in polls 
Should users of this group be able to participate (vote) in polls in this forum?

[edit] Miscellaneous

Can search forum 
Should users of this group be able to search for posts or threads in this forum?
Forum & Posts: Forum Moderators

In this tab, you can manage the moderators for this forum. You will see a listing of all moderators in that forum, you can edit each moderator's permissions, and add a new moderator.

First, enter the username of the moderator you would like to add at the bottom of the moderators tab on the View Forum page. Then, select "Add Moderator." The moderator will be added with default permissions, and you will be taken to a page allowing you to edit the moderator, setting permissions if other than the default. If the default permissions are fine, you can leave this page without saving.

Can edit posts 
Should this moderator be able to edit other users' posts in the forums they moderate?
Can delete posts 
Should this moderator be able to delete other users' posts in the forums they moderate?
Can view IPs 
Should this moderator be able to view IPs of users in the forums they moderate?
Can open/close threads 
Should this moderator be able to open or close threads in the forums they moderate?
Can manage threads (split, move, copy, merge) 
Should this moderator be able to manage threads, including split, move, copy and merge, in the forums they moderate?
Can move threads to another forum this user doesn't moderate 
Should this moderator be able to move threads to a forum that they do not moderate (checked), or should they only be able to move threads to forums they moderate (unchecked).
Forum & Posts: Add New Forum, with all options shown
Create a 
Should the forum you create be a forum that users can post in or a category that contains forums?
Title 
Name for the forum
Description 
Description of the forum to be shown in the forums listings
Parent Forum 
Forum this forum should be placed in. If top level category (not sub category), select "None"
Display Order 
Display order in relation to other forums in its parent forum (if top level category, in relation to other top level categories).

When adding a forum, the additional forum options are hidden by default, but can be expanded by selecting "Show Additional Options" above the quick permissions, and can be hidden again by selecting "Hide Additional Options." When editing a forum, all options are displayed.

Forum Link 
This allows you to make this forum a link to another website, rather than a forum you can post in.
Forum Password 
To protect the forum further, you can password-protect it.
Forum is Active 
Should the forum be shown to users?
Forum is Open 
Should the forum be open for posting? If unchecked, the forum will be "locked" to all posts, regardless of permissions.
Moderate New Posts 
Should new posts in this forum be added to the moderation queue?
Moderate New Threads 
Should new threads in this forum be added to the moderation queue?
Moderate New Attachments 
Should new attachments in this forum be added to the moderation queue?
Moderate After Edit 
After editing a post, should it be added to the moderation queue?
Override User's Selected Style 
Should the selected theme for this forum override the theme the user has selected?
Forum-Specific Style 
If other than default, should the forum have a different style/theme?
Display Method
Should forum rules be shown, and if yes, should they be shown inline or as a link?
Title 
If forum rules will be shown, what should the title be?
Rules 
If forum rules will be shown, what should they be?
Default Date Cut 
What should the default date limit be when viewing forums? Options: Board Default, Last Day, Last 5 Days, Last 10 Days, Last 20 Days, Last 50 Days, Last 75 Days, Last 100 Days, Last Year, The beginning (show all)
Default Sort By 
What should the threads be sorted by, by default? Options: Board Default, Thread Subject, Last Post Time, Thread Starter, Thread Creation Time, Thread Rating, Number of Replies, Number of Views
Default Sort Order
What should the default sort order be in this forum? Options: Board Default, Ascending (A-Z, 0-9), Descending (Z-A, 9-0)
Allow HTML 
Should HTML be allowed in posts in this forum? Javascript is automatically removed.
Allow MyCode 
Should MyCode be allowed in posts in this forum?
Allow Smilies 
Should smilies be shown in posts in this forum?
Allow [img] code 
Should images be shown in posts in this forum?
Allow Post Icons 
Should post icons be allowed to be used in posts in this forum?
Allow Thread Ratings 
Should users be allowed to rate threads (where permissions allow) in this forum?
Show in Forum Jump 
Should this forum be shown in the forum jump?
Make posts in this forum count towards user post counts 
Should posts in this forum count towards users' post counts? Posts will always count towards the board total, regardless of this setting. If editing this setting, post counts will need to be recounted through Recount & Rebuild to reflect the change.

Forum & Posts: Copy Forum

You can copy forum settings or forum permissions to another forum, be it an existing forum or a new forum. You can choose to copy a forum from the Options drop down from the forum listing page, or from the View Forum page.

Source Forum 
What forum should the settings and/or permissions be copied from?
Destination Forum 
What forum should the settings and/or permissions be copied to? To copy to an existing forum, select the forum. To copy to a new forum, select "Copy to new forum"
Copy Forum Settings and Properties 
Should the forum settings and properties be copied? This does not copy posts, just forum settings.
Copy User Group Permissions 
Should forum permissions be copied? If yes, select the user groups to be copied. If no, do not select any user groups.

If copying to a new forum, there are additional settings you must fill out.

Create a 
Should the forum you create be a forum that users can post in or a category that contains forums?
Title 
Name for the forum
Description 
Description of the forum to be shown in the forums listings
Parent Forum 
Forum this forum should be placed in. If top level category (not sub category), select "None"
