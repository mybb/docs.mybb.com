Configuration: Post Icons

The Post Icons page allows management of all post icons on your forum. On the post icons listing page, you can see the post icon itself and the name for the post icon. From here you can also add new post icons and manage existing post icons.

Configuration: Add Post Icon
Name 
The name to recognize the post icon by. It also is the "alt" element for the image.
Image Path 
The relative path (from the MyBB root) to the image to be set for this post icon. Images for post icons must be located in a relative position to the forum root.
Configuration: Add Multiple Post Icons
Configuration: Post Icons

First, you will be shown a page asking for the URL to the directory to be searched for the post icons. This directory must be located in a relative position to the forum root.

You will then be taken to a page showing all post icons in that directory that are not already added to your forum. You are shown the following details:

  • Image
  • Image URL
  • Suggested Name

From here, you can modify the following details:

Name 
If different than the suggested name, you can modify the name for each post icon found.
Add 
You can also select which post icons you would like to add to your forum. All are selected by default, but you can choose to not add post icons by deselecting the checkbox next to the corresponding post icon.

After checking all details to ensure they are correct, you can select "Save Post Icons" at the bottom of the page, submitting the form and adding the smilies to your forum.
