Configuration: Thread Tools
Configuration: Post Tools

This section allows you to manage the thread and post moderator tools for your forum. This feature allows you to combine moderation actions, and make these combined actions, called "mod tools," available in select forums.

On both the thread tools and post tools listing, you will see the title and description for the mod tool.

Configuration: Add Thread Tool

The generic settings for this moderator tool.

Name 
Title to describe this mod tool.
Short Description 
Describes the functions this mod tool performs.
Available in Forums 
Should the mod tool be available in all forums, or select forums? If you choose to select forums, you will be presented with a list of the forums on your board. You can select more than one by holding down CTRL.

Moderator functions to be performed on the thread.

Approve/Unapprove Thread 
Should the thread be approved, unapproved, or have its status toggled (if unapproved, set to approved; if approved, set to unapproved)?
Open/Close Thread 
Should the thread be opened, closed, or have its status toggled (if closed, set to open; if open, set to closed)?
Move Thread 
Should the thread be moved to another forum? If yes, select the forum, if a redirect should be left behind, and how many days it should be left behind (leave blank to never remove the redirect).
Copy Thread 
Should the thread be copied to another forum? If yes, select the forum.
Delete Thread 
Should the thread be removed completely?
Merge Thread 
Should the selected threads in inline moderation be merged?
Delete Poll 
If there is a poll, should it be removed?
Delete redirects 
If the thread has any redirects to it, should they be removed?
New Subject 
If you wish, you can change the subject of the thread. {subject} represents the existing subject, and {username} represents the username of the moderator who is using the mod tool.

Add a new reply to the thread.

Add New Reply 
Post contents; leave blank to not add a new reply.
Reply Subject 
Only applies if a new reply is being made. {subject} represents the existing subject, and {username} represents the username of the moderator who is using the mod tool.
Configuration: Add Post Tool

See General Options above.

Functions to be performed on the selected posts.

Delete posts 
Should the selected posts be removed completely?
Merge posts 
Should the selected posts be merged?
Approve/Unapprove posts 
Should the posts be approved, unapproved, or have their status toggled (if unapproved, set to approved; if approved, set to unapproved)?
Split posts 
Should the posts be split to a new thread in another forum? If yes, either select Split to Same Forum or another forum you would like the posts to be split to.
Close split thread 
Should the new thread be closed?
Stick split thread 
Should the new thread be stuck to the top of the forum?
Unapprove split thread 
Should the new thread be unapproved?
Split thread subject 
Subject of split thread (if splitting posts) - {subject} represents the original subject. Only required if splitting posts.
Add reply to split thread 
Post contents; leave blank to not add a new reply.
Reply Subject 
Only applies if a new reply is being made. {subject} represents the existing subject, and {username} represents the username of the moderator who is using the mod tool.
