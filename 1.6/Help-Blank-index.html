Getting a blank index page?

There are four common reasons listed below for why you might be getting a blank index page.

Another file is called index

Usually your host will give priority to html files when deciding what page to set as your home page so if your receiving a blank or unusual index page you should first check that there is not already a file called index.html or index.htm in your forum directory. If there is then try renaming or deleting the file. This may also be caused by files named home on some servers.

Your hosting does not support Gzip compression

Often if your host does not support Gzip compression and you have it enabled you will get a blank page. Read help:gzip for more information on how to dissable Gzip compression.

There is a problem with your .htaccess file

If the above two solutions don't work then there might be an issue with your .htaccess file, try renaming it to htaccess.txt and see if that helps.

You've not got any templates

View the source of a page on your forum. If you just see this: 

<!-- start: index -->

<!-- end: index -->

It means no templates have been imported. If this is a fresh install, and you have not added any content to your forum yet, reupload a new copy of ./install/resources/mybb_theme.xml, make sure it's uploaded properly and doesn't have a size of 0kb, and install again. This will reinstall the forum from scratch so do not do this if you have added content to your forum. If you have added content to your forum and do not want to reinstall, make a support thread and we'll find the best way of solving the problem.
