Configuration: Help Documents

The Help Documents section of the Admin CP allows you to manage the help documents that users can view when accessing your board. These include basic information on the functionality of your forum, but you can also add your own help documents for whatever reason you see fit.

From the main Help Documents listing page, you will see a full listing of all of your Help Sections and Help Documents. You can see the description for each section or document, and see whether the section or document comes with your MyBB installation or is one you custom created just for your forum.

Configuration: Add Help Document
Section 
The section this help document should be listed under.
Title 
Name to represent this document
Short Description 
A brief statement quickly outlining the contents of this document.
Document 
The full contents of this help document. You can use HTML, but cannot use MyCode.
Display Order 
Order in relation to other help documents in the section.
Enabled 
Should this help document be displayed?
Use Translation 
If a default help document, should this document use the translation in the language files?
Configuration: Add Help Section
Title 
Name to represent this section
Short Description 
Brief statement describing contents of help documents within this section.
Display Order 
Order in relation to other help sections.
Enabled 
Should this help section be displayed?
Use Translation 
If a default help section, should this section use the translation in the language files?
