MyBB 1.4.5 was released on 19 April 2009.

MyBB 1.4.5 is now available on the MyBB website and is a general maintenance and security release.

This release fixes numerous issues with versions released since 1.4.4 with some causing incorrect functionality of MyBB. These bugs have been fixed to provide a more stable version of MyBB for public use.

More information here: http://community.mybb.com/thread-48484.html
