Users & Groups: User Titles

User Titles are displayed under the username in posts when there is no custom user title specified for the user and there is no group user title set. User titles are based on number of posts, and also represent the number of "stars" shown beneath the user's name.

The user title listing includes the user title and the number of posts required for a user to gain that user title. From here, you can edit or delete each user title, or add a new user title.

Users & Groups: Add User Title
Title to Assign
The user title that will be displayed for users with the set number of minimum posts.
Minimum Posts 
The minimum number of posts a user most accumulate before they are assigned this user title.
Number of Stars 
The number of stars that will be displayed based on the minimum posts for this user title.
Star Image 
The image to be displayed for the star image. If this field is empty, the group star image will be used. Use {theme} to specify the image directory for the viewer's current theme.
